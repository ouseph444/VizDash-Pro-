# ⚡ Technical Architecture

VizDash Pro is implemented as a **single-file HTML application**.

```text
Browser
   │
   ├── Data Parsing
   │      └── PapaParse
   │
   ├── Visualization
   │      └── Plotly.js
   │
   ├── Mathematics
   │      ├── MathJax
   │      └── KaTeX
   │
   ├── UI Layer
   │      └── Tailwind CSS
   │
   └── Dashboard Layout
          └── Sortable.js
```

### Benefits

* Zero Installation
* Fully Portable
* Offline Compatible
* Privacy Preserving
* No Server Processing

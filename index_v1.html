Automatic EDA Report

Upload data to generate descriptive statistics, correlations, pairplots, and column-level visuals.

This report profiles each column based on its detected type, then computes descriptive summaries, uncertainty intervals, outlier fences, and relational views between variables. Correlation significance is shown with star notation based on p-values.

Statistical Tests Engine

Recommendation will appear after data upload and column selection.
The test recommender looks at variable types, paired versus independent structure, and rough normality diagnostics. Outputs include the key statistic, uncertainty, an effect size when appropriate, and a plain-English decision card.

Chi-Square Analysis

These workflows compare observed and expected frequencies. Standardized residuals highlight the cells driving association, and effect size is reported with Cramer's V where appropriate.

A/B Testing Suite

The A/B panel combines sample size planning, frequentist significance testing, sequential monitoring, multiple-testing correction, and a Bayesian decision layer for binary experiments.

Likelihood & Information Criteria

Candidate distributions are fit by maximizing the log-likelihood. Information criteria help compare models on fit-versus-complexity tradeoffs, while profile likelihoods visualize uncertainty around each parameter.

Bayesian Inference

The prior builder shows the belief you encode before looking at data. Conjugate models update analytically, while non-conjugate normal-mean sampling is estimated with four Metropolis-Hastings chains and convergence diagnostics.

Time Series Analysis

The time-series workflow sorts observations chronologically, decomposes the series, checks stationarity, explores autocorrelation structure, evaluates lightweight ARIMA and ETS candidates, and produces forecast bands and break-point flags.

Forecasting & Prediction

This panel compares several browser-side predictive models using the same split or cross-validation setup, then ranks them on out-of-sample performance and exposes scenario prediction controls for fitted models.
